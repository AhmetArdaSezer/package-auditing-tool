# Kaynak Listesi

Araştırma sürecinde kullanılan temel teknik dokümanlar, güvenlik standartları ve açık kaynak proje referansları aşağıdadır:

1.  **Python Security & Auditing Libraries**
    * *Red Hat Developer - Find Python Vulnerabilities:* https://www.redhat.com/en/blog/find-python-vulnerabilities
    * *PyPI - pip-audit:* https://pypi.org/project/pip-audit/
    * *OWASP Dep-Scan:* https://owasp.org/www-project-dep-scan/

2.  **Linux Vulnerability Standards & Databases**
    * *NIST National Vulnerability Database (NVD):* https://nvd.nist.gov/
    * *Canonical Ubuntu CVE Tracker:* https://ubuntu.com/security/cves
    * *Debian Security Tracker:* https://security-tracker.debian.org/tracker/
    * *Red Hat CVE Database (cvelib):* https://github.com/RedHatProductSecurity/cvelib

3.  **Competitor & Comparative Analysis**
    * *TuxCare - Best Open Source Security Tools:* https://tuxcare.com/blog/open-source-security-tools/
    * *eSecurity Planet - Top Vulnerability Scanners:* https://www.esecurityplanet.com/networks/open-source-vulnerability-scanners/
    * *Qualysec - Top 10 Open Source Scanners:* https://qualysec.com/open-source-vulnerability-scanners/

4.  **System Auditing & Configuration**
    * *Linux Audit Documentation (auditd):* https://man7.org/linux/man-pages/man8/auditd.8.html
    * *SentinelOne - Linux Security Audit Guide:* https://www.sentinelone.com/cybersecurity-101/cybersecurity/linux-security-audit/
