# Kaynak Listesi

Bu araştırma raporu hazırlanırken aşağıdaki teknik dökümanlar ve veritabanlarından yararlanılmıştır:

1. **NVD (National Vulnerability Database):** ABD hükümetinin standartlara dayalı zafiyet veri deposu. [https://nvd.nist.gov/](https://nvd.nist.gov/)
2. **CycloneDX Standard:** SBOM (Software Bill of Materials) için endüstri standardı olan tam nesne modeli. [https://cyclonedx.org/](https://cyclonedx.org/)
3. **OSV (Open Source Vulnerabilities):** Açık kaynak projeler için dağıtık zafiyet veritabanı ve API servisi. [https://osv.dev/](https://osv.dev/)
4. **OWASP Dependency-Check:** Yazılım bağımlılıklarındaki zafiyetleri tespit etme rehberi. [https://owasp.org/www-project-dependency-check/](https://owasp.org/www-project-dependency-check/)
5. **Python Safety DB:** Python paketlerine özel güvenlik açığı veritabanı projesi. [https://github.com/pyupio/safety-db](https://github.com/pyupio/safety-db)
